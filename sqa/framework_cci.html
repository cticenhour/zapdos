Communication and Contact Information

Introduction

The Communication and Contact Information (CCI) for provides information for contacting the software development team for all aspects of Multiphysics Object Oriented Simulation Environment (MOOSE). This includes information about reporting a problem with a release, asking questions regarding any version of the software, and contributing to the project.

General Communication

This section details how to communicate with the MOOSE development team to get help, report a suspected bug, or request an enhancement.

Please use the MOOSE Discussion forum for all questions and inquires.

Problem Reporting

This section details how to communicate with the MOOSE development team to report a potential problem with a specific release of the application.

If you are having a problem with a specific release of MOOSE please contact the team using the MOOSE Discussion forum and be sure to include the release information with the inquiry.

Contributing

MOOSE and MOOSE-based applications are always improving and changing to meet the needs of the community. In most circumstances changes are welcome. If you are interested in contributing to the project please refer to the following instructions.

Please refer to the Contributing instructions for details regarding how to help improve MOOSE.
