Gnuplot

Output for postprocessors and scalar variables in GNU plot format.

Overview

The GNU output writes postprocessor and scalar variable data in a format suitable for use with gnuplot.

Input Parameters

  • append_restartFalseAppend existing file on restart

    Default:False

    C++ Type:bool

    Controllable:No

    Description:Append existing file on restart

  • extensionpngGNU plot file extension

    Default:png

    C++ Type:MooseEnum

    Options:png, ps, gif

    Controllable:No

    Description:GNU plot file extension

  • file_base_suffixSuffix to add to the file base

    C++ Type:std::string

    Controllable:No

    Description:Suffix to add to the file base

  • use_displacedFalseEnable/disable the use of the displaced mesh for outputting

    Default:False

    C++ Type:bool

    Controllable:No

    Description:Enable/disable the use of the displaced mesh for outputting
