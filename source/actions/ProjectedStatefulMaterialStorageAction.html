ProjectedStatefulMaterialStorageAction

Description

The ProjectedStatefulMaterialStorageAction is the main action in the ProjectedStatefulMaterialStorage system which sets up all necessary objects to project material property components onto nodal or elemental basis functions.

Projected and interpolated old state can be enabled using the use_interpolated_state parameter available in the MaterialPropertyInterface.

Objects set up by the action

The action will set up aux variables for each scalar component of the projected properties listed in "projected_props". The variables will be marked as hidden and will not appear in any outputs.

The type of each projected material property will be determined automatically and a corresponding InterpolatedStatefulMaterial object will be created to reconstitute a material property of the same type from the old state of the projected variables.

The "family" parameter determines whether nodal or elemental basis functions will be used.

Elemental basis functions

A ProjectedStatefulMaterialAux aux kernel of the appropriate type will be added for each scalar material property component (e.g. vector or tensor components) to perform an elemental projection of the property component.

Nodal basis functions

A ProjectedStatefulMaterialNodalPatchRecovery user object of the appropriate type will be created for each material property to prepare data required for nodal patch recovery of each scalar material property component (e.g. vector or tensor components).

A ProjectedMaterialPropertyNodalPatchRecoveryAux aux kernel will be added for each scalar material property component to perform nodal patch recovery using the data collected in the user object.

projected_props

C++ Type:std::vector<MaterialPropertyName>

Controllable:No

Description:Material properties to project for stateful storage

family

Default:LAGRANGE

C++ Type:MooseEnum

Options:LAGRANGE, MONOMIAL, L2_LAGRANGE, L2_HIERARCHIC

Controllable:No

Description:Finite element variable family to project the material properties onto
