NEML2ToMOOSEMaterialProperty

commentnote

Users are NOT expected to directly use this object in an input file. Instead, it is always recommended to use the NEML2 action.

Description

This family of objects assign a NEML2 output variable given by "from_neml2" back to the quadrature points. The data is stored as a MOOSE material property with name given by "to_moose".

Note that if the "neml2_input_derivative" parameter is specified, the derivative of the NEML2 output variable with respect to the input variable is retrieved instead. If the "neml2_parameter_derivative" parameter is specified, the derivative of the NEML2 output variable with respect to the model parameter is retrieved instead.

The following flavors exist:

ClassMOOSE MaterialProperty type
NEML2ToMOOSERealMaterialPropertyReal
NEML2ToMOOSESymmetricRankTwoTensorMaterialPropertySymmetricRankTwoTensor
NEML2ToMOOSESymmetricRankFourTensorMaterialPropertySymmetricRankFourTensor
NEML2ToMOOSEStdVectorMaterialPropertystd::vector<Real>
